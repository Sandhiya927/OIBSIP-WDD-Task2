Sandhiya – Portfolio Website

This is a simple and responsive personal portfolio website built using HTML and CSS.  
It showcases profile, skills, projects, and provides an option to download my resume.

Project Description

The portfolio website includes sections such as:
- About Me  
- Skills  
- Projects  
- Resume Download  
- Contact  

It is designed with a clean layout and responsive structure, suitable for beginners in frontend web development.

Technologies Used

- HTML
- CSS

How to Run the Project

1. Clone or download the repository  
2. Open `index.html` in any web browser
